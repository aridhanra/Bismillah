Halo Semua!

Yang kita pelajari hari ini:

  1. Pendaftaran akun GITHUB
  2. Join ke repository (Accept Invitation)
  3. Membuat repository GIT
  4. Membuat base code (index.html) dan melakukan initial commit
  5. Membuat file html masing-masing
  6. Semuanya melakukan push atas file html yang sudah dibuat
  7. Melakukan PULL untuk melihat semua file yang sudah ada di remote agar ada di laptop masing-masing
